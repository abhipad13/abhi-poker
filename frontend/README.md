# Poker UI

A modern React-based poker game interface built with Tailwind CSS v4 and Vite.

## Features

- **Join/Create Game Screen**: Beautiful casino-themed UI for creating new poker games or joining existing ones
- **Responsive Design**: Works on desktop and mobile devices
- **Modern Tech Stack**: React 18, TypeScript, Tailwind CSS v4, Vite
- **API Integration**: Ready to connect with Spring Boot backend

## Prerequisites

- Node.js 18+ 
- npm or yarn
- Spring Boot backend running on `http://localhost:8080` (for API calls)

## Setup

1. **Install dependencies**:
   ```bash
   npm install
   ```

2. **Environment Configuration**:
   Create an env file as needed:
   - Development (uses Vite proxy; backend on 8080):
     ```env
     VITE_API_BASE=http://localhost:8080
     ```
   - Production when hosted by Spring Boot (same-origin):
     ```env
     # Leave empty to use relative paths and same-origin
     VITE_API_BASE=
     ```

## Running the Application

### Development Mode

You need to run **two terminals** simultaneously:

**Terminal 1 - Tailwind CSS watcher**:
```bash
npm run tw:watch
```

**Terminal 2 - Vite dev server** (proxy to backend on 8080 is configured):
```bash
npm run dev
```

The application will be available at: `http://localhost:5173`

### Production Build

```bash
npm run build
npm run preview
```

### Deploying into Spring Boot static resources

1. Build with same-origin base:
   ```bash
   echo "VITE_API_BASE=" > .env.production
   npm run build
   ```
2. Copy `dist/` contents into your Spring Boot app's `src/main/resources/static/`.
3. Add a Spring MVC forwarder so SPA routes serve `index.html` (example in your backend):
   - forward non-`/api/**` paths to `/index.html`.

```
├── styles.css            # Tailwind input file

```

## API Endpoints

The UI expects these Spring Boot endpoints:

- `POST /api/game/create?managerName={name}` - Create a new game
- `POST /api/game/{gameId}/player?name={name}` - Join an existing game

## Tailwind CSS v4 Setup

This project uses Tailwind CSS v4 with the CLI approach:

- **Input**: `src/styles.css` (contains `@import "tailwindcss"` and custom CSS)
- **Output**: `src/tailwind.css` (generated by CLI, imported in main.tsx)
- **CLI Script**: `npm run tw:watch` watches for changes and rebuilds

## Customization

The casino theme uses CSS custom properties:
- `--felt`: Green felt background color
- `--gold`: Gold accent color for buttons and highlights

Modify these in `src/styles.css` to change the theme.
